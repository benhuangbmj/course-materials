\noindent
{\bf Exit Ticket (Argue).} Score \rule{2em}{0.4pt} 

Solve $\log_5(x+3) - \log_5x = 2$ for $x$.

\begin{enumerate}[label = \textbf{Step \arabic*:}, leftmargin=*]
\item
Combine each side into a single logarithm with the same base.
\vspace{\stretch{1}}
\item
Exponentiate both sides with the appropriate base.
\vspace{\stretch{1}}
\item
Solve the resulted simplified equation.
\vspace{\stretch{1}}
\item Check for extraneous solutions. Note that the input of the log cannot be $0$ or negative as well.
\vspace{\stretch{1}}
\end{enumerate}

\noindent\textbf{Learning Objectives.}
\begin{itemize}
\item
Understand the components of a 3D coordinate system
\item 
Plot the graph of a simple two variable function
\end{itemize}
\noindent\textbf{Problems.}
\begin{enumerate}
\item
Describe how to construct a 2D coordinate system, and how to assign coordinates to a point in the coordinate plane.
\vspace{\stretch{1}}
\item
Describe how you will extend the notion of coordinate system to the 3-dimensional space.
\vspace{\stretch{1}}
\item
Describe what the graph of a function $y=f(x)$ means in the context of a 2D coordinate system.
\vspace{\stretch{1}}
\clearpage
\item Extend the notion of the graph to the 2-variable function $z = f(x,y)$.
\vspace{\stretch{1}}
\item 
Plot the graph of $f(x,y) = 1$ by hand in a 3D coordinate space.
\vspace{\stretch{1}}
\end{enumerate}
